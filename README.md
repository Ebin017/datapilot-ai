# DataPilot AI

> An Agentic AI platform that automates the end-to-end data science workflow using CrewAI.

## Vision

DataPilot AI helps businesses transform raw datasets into actionable business insights through an AI-powered data science consulting workflow.

Instead of manually performing data cleaning, exploratory data analysis, model selection, explainability, and reporting, users upload a dataset and receive a complete analysis package.

## Version 1 Goals

- Upload CSV datasets
- Analyze dataset structure
- Generate an AI analysis plan
- Perform automated data cleaning
- Run exploratory data analysis
- Train multiple machine learning models
- Compare model performance
- Explain predictions using SHAP
- Generate business insights
- Produce an executive PDF report
